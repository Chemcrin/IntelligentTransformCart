## 1. IMU 传感器变量

**来源：** `board_app.h` / `board_app.c`（V3 GPT 代码，`ATK_JY_NewVer_GPT` 工程）  
**类型：** 全局变量，`volatile` 修饰，供中断和主循环共享  
**物理层：** I2C1 → JY61P

### 1.1 全局变量（推荐使用 g_ 前缀）

```c
// ===== 姿态角 =====
extern volatile float  g_imu_car_yaw_est_deg;    /* 融合偏航角，deg */
extern volatile float  g_imu_car_pitch_deg;       /* 俯仰角，deg */
extern volatile float  g_imu_car_roll_deg;        /* 横滚角，deg */

// ===== 角速度 =====
extern volatile float  g_imu_gyro_yaw_dps;        /* 偏航角速度，deg/s */

// ===== 加速度 =====
extern volatile float  g_imu_acc_forward_g;       /* 前向加速度，g */
extern volatile float  g_imu_acc_right_g;         /* 右向加速度，g */
extern volatile float  g_imu_acc_down_g;          /* 向下加速度，g */

// ===== 温度与状态 =====
extern volatile float  g_imu_temperature_c;        /* 温度，℃ */
extern volatile uint8_t g_imu_valid;               /* 数据有效标志 */
extern volatile uint8_t g_imu_calibrated;          /* 校准完成标志 */
```

### 1.2 兼容变量名（无 g_ 前缀，供旧代码过渡）

| 兼容名 | 映射到 | 说明 |
|---|---|---|
| `car_yaw_est_deg` | `g_imu_car_yaw_est_deg` | 建议新代码用 g_ 前缀 |
| `car_pitch_deg` | `g_imu_car_pitch_deg` | |
| `car_roll_deg` | `g_imu_car_roll_deg` | |
| `gyro_yaw_dps` | `g_imu_gyro_yaw_dps` | |
| `acc_forward_g` | `g_imu_acc_forward_g` | |
| `acc_right_g` | `g_imu_acc_right_g` | |
| `acc_down_g` | `g_imu_acc_down_g` | |
| `temperature_c` | `g_imu_temperature_c` | |
| `imu_valid` | `g_imu_valid` | |
| `imu_calibrated` | `g_imu_calibrated` | |

### 1.3 已移除的旧字段

V2.3 及之前版本中存在，V3 已移除的变量名（不再使用）：

| 移除的旧名                                      | 移除原因                     |
| ------------------------------------------ | ------------------------ |
| `imu.ax_g` / `imu.ay_g` / `imu.az_g`       | 改为独立全局变量 `g_imu_acc_*_g` |
| `imu.gx_dps` / `imu.gy_dps` / `imu.gz_dps` | 代码只保留 yaw 轴角速度           |
| `imu.error_count`                          | 代码中已无此变量                 |
| `imu.last_update_ms`                       | 代码中已无此变量                 |

## 2. 激光测距变量

**来源：** `board_app.h` / `board_app.c`（V3 GPT 代码）  
**物理层：** I2C1 → ATK-MS53L1M / VL53L1X

### 2.1 全局变量

```c
// ===== 距离数据 =====
extern volatile uint16_t g_laser_raw_distance_mm;         /* 最新原始距离，mm */
extern volatile uint16_t g_laser_filtered_distance_mm;     /* 滤波后稳定距离，mm */
extern volatile uint16_t g_laser_distance_mm;              /* 对外输出距离，mm */

// ===== 状态标志 =====
extern volatile uint8_t  g_laser_valid;                   /* 距离数据是否有效 */
extern volatile uint8_t  g_laser_ok;                      /* 模块是否在线 */
extern volatile uint8_t  g_laser_is_warning;              /* 预警标志，< 600 mm */
extern volatile uint8_t  g_laser_is_slow;                 /* 减速标志，< 300 mm */
extern volatile uint8_t  g_laser_is_too_close;            /* 停车标志，< 150 mm */

// ===== 时间 =====
extern volatile uint32_t g_laser_last_valid_ms;           /* 最近有效距离的时间戳 */
```

### 2.2 预警等级对照

```c
// 三个标志位组合 → 等级
// g_laser_is_warning  | g_laser_is_slow  | g_laser_is_too_close  →  等级
//        0            |        0         |         0             →  0（正常）
//        1            |        0         |         0             →  1（预警）
//        1            |        1         |         0             →  2（减速）
//        1            |        1         |         1             →  3（停车）
```

## 3. 桥接层 API

**来源：** `sensor_bridge.h` / `sensor_bridge.c`  
**作用：** 将传感器层变量转换为运动控制层需要的单位（rad/m）

### 3.1 函数接口

```c
// ===== IMU =====
float  JY61P_GetYawRad(void);           // g_imu_car_yaw_est_deg * DEG_TO_RAD → rad
float  JY61P_GetGyroZRadS(void);        // g_imu_gyro_yaw_dps * DEG_TO_RAD → rad/s
float  JY61P_GetPitchDeg(void);         // g_imu_car_pitch_deg → deg（直传）
float  JY61P_GetRollDeg(void);          // g_imu_car_roll_deg → deg（直传）
bool   JY61P_IsDataValid(void);         // g_imu_valid → bool
bool   JY61P_IsCalibrated(void);        // g_imu_calibrated → bool

// ===== 激光 =====
bool   Laser_GetDistance(float *d);     // g_laser_filtered_distance_mm / 1000 → m
uint16_t Laser_GetRawDistanceMm(void);  // g_laser_raw_distance_mm → mm（直传）
uint8_t  Laser_GetWarningLevel(void);   // 三标志组合 → 0/1/2/3
bool   Laser_IsOnline(void);            // g_laser_ok → bool
```

### 3.2 快照结构体

```c
typedef struct {
    float    yaw_rad;           /* 偏航角，单位 rad       ← g_imu_car_yaw_est_deg */
    float    gyro_z_rads;       /* 角速度，单位 rad/s     ← g_imu_gyro_yaw_dps */
    float    pitch_deg;         /* 俯仰角，单位 deg       ← g_imu_car_pitch_deg */
    float    roll_deg;          /* 横滚角，单位 deg       ← g_imu_car_roll_deg */
    float    distance_m;        /* 激光距离，单位 m       ← g_laser_filtered_distance_mm */
    bool     imu_valid;         /* IMU 有效               ← g_imu_valid */
    bool     imu_calibrated;    /* IMU 已校准             ← g_imu_calibrated */
    bool     laser_valid;       /* 激光有效               ← g_laser_valid */
    uint8_t  laser_warning;     /* 预警等级 0-3           ← 三标志组合 */
} SensorSnapshot_t;

// 获取快照
void Sensor_GetSnapshot(SensorSnapshot_t *snap);
```

### 3.3 内部缓存变量

```c
// sensor_bridge.c 内部使用，不对外暴露
static SensorSnapshot_t s_snap_cache;    /* 快照缓存 */
static bool             s_snap_valid;    /* 缓存有效性 */
```

## 5. 电机 Modbus 变量

**来源：** 麦轮运动控制系统设计文档 + ZDT_X57_V2 Modbus 手册  
**物理层：** USART1 → RS485 → Modbus-RTU → ZDT-X42-V2 电机驱动器

### 5.1 电机地址

```c
#define MOTOR_ADDR_FL    0x01      /* 前左轮 FL */
#define MOTOR_ADDR_FR    0x02      /* 前右轮 FR */
#define MOTOR_ADDR_RL    0x03      /* 后左轮 RL */
#define MOTOR_ADDR_RR    0x04      /* 后右轮 RR */
#define MOTOR_ADDR_BROAD 0x00      /* 广播地址 */
```

### 5.2 Modbus 寄存器地址

```c
// ===== 写寄存器 =====
#define ZDT_REG_SPEED_CTRL    0x00E6   /* 速度模式控制（4 个寄存器） */
//   Reg1 (0x00E6): 方向 CW=0x0000 / CCW=0x0001
//   Reg2 (0x00E7): 加速度 RPM/s, uint16
//   Reg3 (0x00E8): 速度 RPM, uint16
//   Reg4 (0x00E9): 同步标志 0x0000=立即 / 0x0100=缓存

#define ZDT_REG_ENABLE        0x00E0   /* 使能控制（2 个寄存器） */
//   Reg1 (0x00E0): 0xAB01=使能 / 0xAB00=失能
//   Reg2 (0x00E1): 同步标志

#define ZDT_REG_STOP          0x00FE   /* 立即停止（1 个寄存器） */
//   Reg1 (0x00FE): 0x9800=紧急停止

#define ZDT_REG_SYNC_TRIGGER  0x00FF   /* 同步触发（1 个寄存器） */
//   Reg1 (0x00FF): 0x6600=触发所有缓存

#define ZDT_REG_CHANGE_ID     0x00A2   /* 修改设备地址（2 个寄存器） */

// ===== 读寄存器 =====
#define ZDT_REG_SPEED_FB      0x0044   /* 实时转速反馈（2 个寄存器） */
//   Reg1 (0x0044): 符号 0=正 / 1=负
//   Reg2 (0x0045): 转速绝对值, uint16

#define ZDT_REG_SYSTEM_STATUS 0x0078   /* 系统状态参数（17 个寄存器） */
```

### 5.3 Modbus 功能码

```c
#define MODBUS_FUNC_READ  0x04   /* 读输入寄存器 */
#define MODBUS_FUNC_WRITE_SINGLE 0x06  /* 写单个寄存器 */
#define MODBUS_FUNC_WRITE_MULTI  0x10  /* 写多个寄存器 */
```

### 5.4 电机管理变量


## 7. 跨模块变量对照表

### 7.1 从传感器层到运动控制层

| 传感器层全局变量 | 桥接层 API | 运动控制层使用 | 单位变化 |
|---|---|---|---|
| `g_imu_car_yaw_est_deg` | `JY61P_GetYawRad()` | 航向保持、转向 | deg → rad |
| `g_imu_gyro_yaw_dps` | `JY61P_GetGyroZRadS()` | 角速度阻尼 | deg/s → rad/s |
| `g_imu_car_pitch_deg` | `JY61P_GetPitchDeg()` | 姿态安全检测 | deg 不变 |
| `g_imu_car_roll_deg` | `JY61P_GetRollDeg()` | 姿态安全检测 | deg 不变 |
| `g_laser_filtered_distance_mm` | `Laser_GetDistance()` | 距离限速、沿墙 | mm → m |
| `g_laser_is_too_close` | `Laser_GetWarningLevel()` | 紧急停车 | 标志 → 等级 |
| `g_imu_valid` | `JY61P_IsDataValid()` | 控制有效性判断 | uint8 → bool |
| `g_imu_calibrated` | `JY61P_IsCalibrated()` | 初始化状态检查 | uint8 → bool |

### 7.2 从运动控制层到 Modbus

| 运动控制变量 | 桥接转换 | Modbus 寄存器 |
|---|---|---|
| `ChassisVel_t.vx` | 逆运动学 → `WheelRpm_t` | `0x00E6 Reg3`（转速） + `Reg1`（方向） |
| `ChassisVel_t.vy` | 同上 | 同上 |
| `ChassisVel_t.omega` | 同上 | 同上 |
| 同步触发命令 | — | `0x00FF` 广播 `0x6600` |
| 紧急停止 | — | `0x00FE` 广播 `0x9800` |

### 7.3 物理层引脚

| 外设 | 传感器层（V3） | 运动控制层 | 备注 |
|---|---|---|---|
| I2C1 | PB6(SCL) / PB7(SDA) | 不直接使用 | JY61P + VL53L1X 共用 |
| USART1 | PA9(TX) / PA10(RX) | Modbus 通信（经 RS485） | 电机控制链路 |
| USART2 | PA2(TX) / PA3(RX) | 树莓派通信（经 CH340N） | 暂不涉及运动控制 |
