# 智能物流小车

基于 STM32 + 树莓派的智能物流小车项目，集成多种传感器与机械结构，实现自主导航与物料搬运功能。

## 目录结构

```
├── 代码/                 # STM32固件代码、传感器驱动、技术手册
│   ├── 陀螺仪uart/       # JY61P陀螺仪UART驱动代码
│   ├── vibe_coding/      # 姿态模块、树莓派视觉、麦轮框架设计资料
│   └── *.md              # 技术手册与设计文档
├── pi/                   # 树莓派3D模型 (SLDPRT/SLDASM/STEP)
├── PCB/                  # PCB设计文件
├── 三合一/               # 三合一模块相关文件
├── 历史建模存档/         # 旧版3D模型备份
├── 项目资料包/           # 项目参考资料
├── *.SLDPRT / *.SLDASM   # SolidWorks 机械结构模型
├── LICENSE               # MIT 许可证
└── README.md             # 本文件
```

## 硬件平台

- **主控**：STM32 系列微控制器
- **协处理器**：树莓派 4B
- **传感器**：JY61P 陀螺仪、ATK-MS53L1M 激光测距
- **执行机构**：ZDT X57 V2 步进闭环电机（Modbus-RTU）
- **机械结构**：底盘、料盘、机械臂、方形料框

## 软件架构

- **STM32 固件**：传感器数据采集、电机控制、I2C/UART 通信
- **树莓派端**：视觉识别、上位机调度
- **通信协议**：I2C (TCA9548A)、UART、Modbus-RTU
